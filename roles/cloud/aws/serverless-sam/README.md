serverless-sam
=========

## これは何？

serverless 開発環境を構築する playbook です。

- docker / docker-compose / ツール類の導入
- dynamodb-local / admin の導入
- sam の導入

Usage
-------

```
cd /opt/docker/dynamodb-local/
docker-compose up -d
```
