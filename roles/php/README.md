php
=========

## これは何？

phpとphpモジュールをインストールするplaybookです。

- レポジトリの追加
- PHPインストール
- 各種モジュールインストール
- php.ini設定

## 変数

php_varに導入するphpのバージョンを定義して下さい。  
php_modulesに導入するモジュールを定義して下さい。

```
---
  php_var: 73
  php_modules:
    - "php"
    - "php-pdo"
    - "php-mbstring"
    - "php-gd"
#    - "php-curl"
#    - "php-pear"
#    - "php-bcmath"
#    - "php-gmp"
#    - "php-intl"
    - "php-mysqlnd"
```
