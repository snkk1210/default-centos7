common
=========

## これは何？

基本的なOS設定用のplaybookです。

- 開発パッケージインストール
- 各種パッケージアップデート
- epelリポジトリインストール
- selinuxライブラリインストール
- selinux無効化
- firewalld無効化
- 運用パッケージインストール
- ホストネーム設定
- カーネルパラメータ設定
- limit.conf設定
- timezone設定
- history設定
- /etc/hostsにホストネームを追加
